# Finsight Frontend

Finsight is a personal finance tracking web application that helps users monitor expenses, analyze spending habits, and gain financial insights through interactive dashboards.

## Features

- User signup/login/logout
- Secure authentication
- Forgot/reset password
- Dashboard overview
- Add/Edit/Delete expenses
- Expense categorization
- Analytics dashboard
- Monthly spending trends
- Weekly comparisons
- Category summaries
- Personalized financial insights
- Token expiry auto logout
- Responsive design

---

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Recharts
- Vercel (Deployment)

---

## Installation

Clone repository:

```bash
git clone https://github.com/meedaah19/FinSight-Frontend
cd react-ts


npm install

Create .env:
VITE_API_BASE_URL=https://your-backend-url.onrender.com

Run locally
npm run dev

 ```
